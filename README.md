# Assignment 1 

Welcome to the first assignment for the Next Level Web Development course!

This assignment contains two blog posts and seven coding problems solutions in a single file named solutions.ts.
The problems cover concepts of basic javascript functions which are included in typescript as well and also covers types, interfaces, classes and generic functions in typescript.

## Project Structure
- solutions.ts: Contains all the coding problem solutions.
- blog-1.md: Contains the first blog post about `unknown` vs `any`
- blog-2.md: Contains the second blog post about `Pick` and `Omit` utility types.
